Welcome to FlosRocketBricks

 Spaceflight related MOCs out of building blocks

A website with instructions and digital files of space MOCs completely for free!
Most of these MOCs can be completely build out of Lego parts.
You can expect me to be adding MOCs every few days :)

A new feature: online instructions!

A new feature is available for (some) new MOCs: online instructions!
You can turn the model at every step making it much easier to build and it is much less work for me compared to pdf instructions :)
If you find this button on an .io file then the new feature is available for that file.
The first MOC that has this feature available is the Phantom Express.
If you want to have this feature enabled for other MOCs or have ideas for improvements just contact me :)
